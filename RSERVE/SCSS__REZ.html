SASS или же SCSS это препроцессор, который служит для быстрого написания CSS стилей. Благодаря SASS вы можете добавить переменные, функции, наследование и многое другое к обычному документу CSS.

Lesson 1.
Переменные.

L1. Переменные.

Функция переменных нужна для того, чтобы редактировать свойство в CSS при помощи нее, а не скролить весь файл в поисках нужного значения у искомого свойтсва.

SCSS

CSS

  • Сначала прописываем знак доллара $.
  • Затем без пробелов прописывается имя переменной(любое).
  • После чего прописывается его значение или свойство.

L3. Вложенность стилей.

Lesson 4.
Миксины.

L4. Миксины.

Миксин - некий шаблон с заранее заданными стилями и свойствами, которые можно повторно использовать, чтобы не прописывать одни и те же стили каждый раз. Прописывается он вначале файла стилей или в отдельном файле.

  • Затем прописываются необходимые стили.

  • Подключается миксин при помощи at-правила @include.

  • Далее внутри прописывается @include и имя миксина.

Аргументы в миксинах.

При определении миксина, аргументы пишутся как имена переменных, разделённые запятыми внутри круглых скобок сразу после имени. Затем, при подключении миксина, значения могут быть переданы аналогичным образом.
Можно прописывать значения для всех заданных аргументов не используя само название аргумента, в таком случае переменные задаются в порядке очереди самих аргументов через запятую, поэтому аргументы , которые могут в дальнейшем меняться лучше прописать вначале.

Lesson 7.
Операторы условия, циклы и @content.

L7. Операторы условия, циклы и @content.

Условия.

Для выполнения условия необходимо вызвать оператор @if указывая в скобках само условие применения свойств.
В данном примере условие выполняется через значение переменной больше либо равно, но есть и другие:
  • >(>=) - больше(больше либо равно)
  • <(<=) - меньше(меньше либо равно)
  • == - равно
  • != - не равно
    • Оператор @else вызывается, если необходимо примененить свойств в случае невыполнения условия оператором @if.
